ทุกอย่างที่ต้องใช้ในการประชุม

ถอดเสียงสด สรุป AI และ export ครบจบในเครื่องเดียว

🎙️

ถอดเสียงเรียลไทม์ ไทย/อังกฤษ

ถอดเสียงเป็นข้อความสดขณะประชุม รองรับไทย อังกฤษ และแบบผสม พร้อมแยกผู้พูดและ timestamp

📝

สรุปประชุมด้วย AI

สร้าง summary, key takeaways และ action items — เลือก template ประชุมได้ 6 แบบ หรือสร้างเองได้

💬

ถามต่อจาก transcript

ถามคำถามเกี่ยวกับการประชุมย้อนหลังได้ทันที AI ตอบพร้อมอ้างอิง timestamp

⭐

Mark Moment สำคัญ

กด bookmark จุดสำคัญด้วยคีย์ลัด Cmd/Ctrl+Shift+M เพื่อให้ AI notes เน้นจุดนั้นเป็นพิเศษ

📤

Export หลายรูปแบบ

ส่งออกเป็น Markdown, Plain text, JSON หรือ SRT subtitle พร้อมรูปแบบสำหรับ Google NotebookLM

🔒

เก็บข้อมูลในเครื่องคุณ

ไม่มีเซิร์ฟเวอร์กลาง ประวัติการประชุมเก็บใน Chrome storage บนเครื่องคุณเท่านั้น

📥

นำเข้า transcript จากไฟล์ v1.2.0

Import บทถอดเสียงจากไฟล์ SRT, VTT, TXT หรือ JSON เพื่อสรุป AI notes จากบันทึกที่มีอยู่แล้ว — พร้อม Workspace ดีไซน์ใหม่ที่อ่านง่ายขึ้น

เริ่มใช้งาน

ใช้งานได้ใน 3 ขั้นตอน

ติดตั้งครั้งเดียว ตั้งค่า API key แล้วเริ่มประชุมได้เลย

1

ติดตั้ง extension

เพิ่ม "Zoom Meeting Transcriber By OATz" จาก Chrome Web Store แล้วปักหมุดไอคอนไว้บน toolbar

2

ใส่ Gemini API Key

ขอ API key ฟรีจาก Google AI Studio แล้ววางในแท็บ Settings — เก็บไว้ในเครื่องคุณเท่านั้น

Settings → Gemini API Key
3

เข้าประชุมแล้วกด Start

เปิดแท็บ Zoom แล้วกด Start เพื่อเริ่มถอดเสียง จบประชุมกด Generate สรุป AI notes ทันที

🔑
ต้องใช้ Gemini API Key ของตัวเอง (BYOK) — Extension นี้ไม่มี backend หรือเซิร์ฟเวอร์กลาง จึงเรียก Gemini API โดยตรงจากเครื่องคุณผ่าน API Key ส่วนตัว
Workspace

ศูนย์กลางจัดการประชุมย้อนหลัง

เปิดหน้าเต็มจอจาก History → “Open Workspace” เพื่อดู แก้ และค้นทุกอย่างของการประชุมในที่เดียว

📄

บทถอดเสียง + ค้นหา

อ่านบทถอดเสียงเต็ม เรียงตามเวลา แยกผู้พูด ค้นหาข้อความ และแก้ชื่อผู้พูดย้อนหลังได้

📝

AI Notes + Scratchpad

สร้างและแก้ไขสรุป AI Notes เอง พร้อมช่อง Scratchpad สำหรับจดส่วนตัว และมีประวัติเวอร์ชันให้กู้คืน

✅

Action Items

ดึงงานที่ต้องทำจากบทถอดเสียงอัตโนมัติ พร้อมผู้รับผิดชอบ กำหนดส่ง สถานะ และลิงก์กลับไปยังหลักฐาน

💬

Highlights + Ask AI

ดูจุดสำคัญที่ปักหมุดไว้ และถามคำถามเกี่ยวกับการประชุมได้ทันที AI ตอบพร้อมอ้างอิง timestamp

📥

Import & Export

นำเข้าบทถอดเสียงจากไฟล์ SRT / VTT / TXT / JSON และส่งออกเป็น Markdown, Plain text, JSON หรือ SRT

🗂️

จัดการรายการประชุม

แถบข้างรวมประชุมทั้งหมด กรอง/เรียงลำดับ จัดกลุ่มตามวัน (วันนี้/เมื่อวาน) และลบประชุมที่ไม่ต้องการได้

✨
ดีไซน์ใหม่ใน v1.2.0 — ยกเครื่องลำดับชั้นหน้า Workspace ให้พื้นผิวแยกชั้นชัดเจน typography และฟอนต์ตรงกับ popup อ่านง่ายทั้งโหมดสว่างและมืด
ความเป็นส่วนตัว

ข้อมูลของคุณ อยู่ในเครื่องคุณ

ออกแบบโครงสร้างแบบ Local-first ไม่มีเซิร์ฟเวอร์กลางของเราคอยเก็บข้อมูลประชุมของคุณ

✓ ไม่มีเซิร์ฟเวอร์กลาง — ไม่มีการส่งข้อมูลไปที่เซิร์ฟเวอร์ของเรา
✓ เสียงและประวัติการประชุมเก็บใน Chrome storage บนเครื่องคุณเท่านั้น
✓ API key เก็บแบบปลอดภัยในเครื่อง ไม่ถูกส่งไปที่อื่น
! เสียงถูกส่งตรงไปที่ Google Gemini API เพื่อถอดเสียง ตามข้อตกลงบริการของ Gemini
คำถามที่พบบ่อย

ไขข้อสงสัยการใช้งาน

ต้องเสียเงินไหม

ตัว Extension ใช้งานฟรีทั้งหมด ค่าใช้จ่ายเดียวคือ Gemini API ซึ่ง Google มี Free tier ให้ใช้อยู่แล้ว

เกี่ยวข้องกับ Zoom AI Companion ไหม

ไม่เกี่ยวข้องกัน Extension นี้พัฒนาขึ้นโดยอิสระ โดยใช้ Google Gemini เป็นเครื่องมือถอดเสียงแทน

รองรับภาษาอะไรบ้าง

รองรับ ภาษาไทย ภาษาอังกฤษ และแบบผสม Thai-English ทั้งการถอดเสียงและสรุป AI Notes

ถ้าเน็ตหลุดกลางประชุม เสียงจะหายไหม

ไม่หาย — ช่วงที่ถอดเสียงไม่สำเร็จเพราะเน็ตหลุดจะถูกเว้นเก็บไว้ในบัฟเฟอร์ และถอดเสียงซ้ำอัตโนมัติเมื่อกลับมาออนไลน์

Workspace คืออะไร เปิดยังไง

Workspace คือหน้าเต็มจอสำหรับจัดการประชุมย้อนหลัง รวมบทถอดเสียง AI Notes งานที่ต้องทำ และ Ask AI ไว้ในที่เดียว เปิดได้จากแท็บ History ในตัว extension แล้วกดปุ่ม “Open Workspace” ที่การประชุมที่ต้องการ

นำเข้าบทถอดเสียงเก่าที่มีอยู่แล้วได้ไหม

ได้ — ใน Workspace รองรับการ Import ไฟล์ SRT, VTT, TXT และ JSON เพื่อนำบทถอดเสียงที่มีอยู่แล้วเข้ามาสรุป AI Notes, ดึง Action items หรือถาม AI ต่อได้ทันที โดยไม่ต้องอัดใหม่

Action items และ Ask AI ทำงานยังไง

Action items ดึงงานที่ต้องทำจากบทถอดเสียงอัตโนมัติ พร้อมผู้รับผิดชอบ กำหนดส่ง และสถานะ กดดูหลักฐานต้นทางในบทถอดเสียงได้ ส่วน Ask AI ให้ถามคำถามเกี่ยวกับการประชุมนั้นได้อิสระ โดย AI ตอบพร้อมอ้างอิง timestamp ที่มาของคำตอบ

แก้ AI Notes เองได้ไหม ถ้าแก้ผิดจะย้อนกลับได้ไหม

ได้ทั้งสองอย่าง — แก้ไข AI Notes ได้เองเต็มที่ พร้อมช่อง Scratchpad สำหรับจดส่วนตัวแยกต่างหาก และทุกครั้งที่สร้างหรือแก้โน้ต ระบบจะเก็บเป็นเวอร์ชันไว้ให้ย้อนดูและกู้คืน (Revisions) ได้

พร้อมเริ่มประชุมแบบมี AI ช่วยจดแล้วหรือยัง
