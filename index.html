ทุกอย่างที่ต้องใช้ในการประชุม

ตั้งแต่ถอดเสียงสด สรุปด้วย AI ไปจนถึง export หลายรูปแบบ — ครบในที่เดียว

🎙️

ถอดเสียงเรียลไทม์ ไทย/อังกฤษ

ถอดเสียงเป็นข้อความสดขณะประชุม รองรับไทย อังกฤษ และแบบผสม Thai-English พร้อมแยกผู้พูดและ timestamp จริง

📝

สรุปประชุมด้วย AI

กด Generate เพื่อสร้าง summary, key takeaways, decisions และ action items — เลือก template ตามประเภทประชุมได้ 6 แบบ หรือสร้างเองได้

💬

ถามต่อจาก transcript

ถามคำถามเกี่ยวกับประชุมย้อนหลังได้ทันที เช่น "ใครพูดถึง deadline บ้าง" — AI ตอบพร้อมอ้างอิง timestamp

⭐

Mark moment สำคัญ

กด Mark moment (หรือ Cmd/Ctrl+Shift+M) เพื่อ bookmark จุดสำคัญระหว่างประชุม พร้อมเลือก label ให้ AI notes เน้นจุดนั้นเป็นพิเศษ

📤

Export หลายรูปแบบ

ส่งออกเป็น Markdown, Plain text, JSON หรือ SRT subtitle พร้อมรูปแบบพิเศษสำหรับ Google NotebookLM

🔒

เก็บข้อมูลในเครื่องคุณ

ไม่มีเซิร์ฟเวอร์กลาง ประวัติการประชุมทั้งหมดเก็บใน Chrome storage ของคุณเท่านั้น ค้นหาและจัดการย้อนหลังได้ทุกเมื่อ

เริ่มใช้งาน

ใช้งานได้ใน 3 ขั้นตอน

ติดตั้งครั้งเดียว ตั้งค่า API key แล้วเริ่มประชุมได้เลย

1

ติดตั้ง extension

เพิ่ม "Zoom Meeting Transcriber By OATz" จาก Chrome Web Store แล้วปักหมุดไอคอนไว้บน toolbar

2

ใส่ Gemini API Key

ขอ API key ฟรีจาก Google AI Studio แล้ววางในแท็บ Settings — ใช้ครั้งเดียว เก็บไว้ในเครื่องคุณเท่านั้น

Settings → Gemini API Key
3

เข้าประชุมแล้วกด Start

เปิดแท็บ Zoom แล้วกด Start เพื่อเริ่มถอดเสียง จบประชุมกด Generate เพื่อสรุป AI notes ทันที

🔑
ต้องใช้ Gemini API Key ของตัวเอง (BYOK) — extension นี้ไม่มี backend หรือเซิร์ฟเวอร์กลาง จึงต้องใช้ API key ส่วนตัวจาก Google AI Studio (มี free tier ให้ใช้) การถอดเสียงและสรุปทุกครั้งเรียก Gemini API ตรงจากเครื่องคุณ
ความเป็นส่วนตัว

ข้อมูลของคุณ อยู่ในเครื่องคุณ

extension นี้ออกแบบมาแบบ local-first ไม่มีเซิร์ฟเวอร์กลางของเราคอยเก็บข้อมูลประชุมของคุณ

✓ ไม่มีเซิร์ฟเวอร์กลาง — ไม่มีการส่งข้อมูลไปที่เซิร์ฟเวอร์ของเรา
✓ เสียงและประวัติการประชุมเก็บอยู่ใน Chrome storage บนเครื่องคุณเท่านั้น
✓ API key ของคุณเก็บในเครื่อง ไม่ถูกส่งไปที่อื่น
! เสียงถูกส่งไปที่ Google Gemini API โดยตรงเพื่อถอดเสียง ตาม Gemini API Terms
คำถามที่พบบ่อย

ยังสงสัยอยู่ใช่ไหม

ต้องเสียเงินไหม

ตัว extension ใช้งานฟรีทั้งหมด ค่าใช้จ่ายเดียวคือ Gemini API ซึ่ง Google มี free tier ให้ใช้อยู่แล้ว เหมาะกับการประชุมทั่วไป

เกี่ยวข้องกับ Zoom AI Companion ไหม

ไม่เกี่ยวข้องกัน extension นี้พัฒนาแยกต่างหากจาก Zoom โดยสิ้นเชิง ใช้ Google Gemini เป็นเครื่องมือถอดเสียงและสรุปแทน

รองรับภาษาอะไรบ้าง

รองรับภาษาไทย อังกฤษ และแบบผสม Thai-English เลือกได้ทั้งฝั่งถอดเสียงและฝั่งสรุป AI notes

แยกผู้พูดได้แม่นแค่ไหน

ระบบแยก speaker เป็น label ทั่วไป (Speaker 1, Speaker 2) และมีฟีเจอร์ทดลอง MFCC fingerprinting แบบ local-only ช่วยให้ label เสถียรขึ้นภายในการประชุมเดียวกัน แต่ไม่ใช่การจดจำเสียงข้ามการประชุมหรือระบุตัวตนจริง

ถ้าเน็ตหลุดกลางประชุม เสียงจะหายไหม

ไม่หาย — ช่วงที่ถอดเสียงไม่สำเร็จเพราะเน็ตหลุดหรือ API error จะถูกเก็บไว้ในบัฟเฟอร์และถอดเสียงซ้ำอัตโนมัติเมื่อกลับมาออนไลน์

พร้อมเริ่มประชุมแบบมี AI ช่วยจดแล้วหรือยัง
