ประวัติการอัปเดต และฟีเจอร์ใหม่

ติดตามการพัฒนา ฟีเจอร์ใหม่ การปรับปรุงประสิทธิภาพ และการแก้ไขทั้งหมดตามลำดับเวอร์ชัน

NEW ฟีเจอร์ใหม่ IMP ปรับปรุงประสิทธิภาพ FIX แก้ไขข้อผิดพลาด BREAKING การเปลี่ยนแปลงสำคัญ
v1.1.0 Latest
20 กรกฎาคม 2026
About Tab, Onboarding & UI Cleanups
  • NEW เพิ่มแท็บ About ใหม่ — รวมเวอร์ชัน ข้อมูลความเป็นส่วนตัว ลิงก์ช่วยเหลือ และคู่มือการใช้งานไว้ในที่เดียว
  • IMP ย้ายปุ่มช่วยเหลือและคำอธิบาย — ย้ายจากแถบเมนูด้านบนเข้าสู่แท็บ About เพื่อให้ UI สะอาดและไม่รบกวนสายตาขณะประชุม
  • NEW First-run Onboarding Guide — เพิ่มหน้าแนะนำการตั้งค่า Gemini API Key และวิธีใช้งานเบื้องต้นสำหรับผู้เปิดใช้ครั้งแรก
v1.0.1
19 กรกฎาคม 2026
Prompt Precision & Store Review Fixes
  • FIX ปรับปรุงความแม่นยำของ Prompt — ปรับแต่ง AI Prompt ภาษาไทยและอังกฤษ เพื่อเก็บคำตอบรับสั้นๆ และประโยคตัดบทในการประชุมได้แม่นยำยิ่งขึ้น
  • FIX แก้ไข Race Condition ใน Local Storage — Serialize ลำดับการบันทึกข้อมูลการประชุมแบบเรียลไทม์ ป้องกันปัญหาข้อมูลชนกันเมื่อมีการพูดพร้อมกัน
  • IMP Export & Copy Cleanups — ปรับเมนูการส่งออกให้กระชับและตรงไปตรงมา เน้นรูปแบบ Plain text, Markdown และ Slack-ready
v1.0.0

Stable 1.0 Official Release
  • NEW Preact & Tailwind UI Architecture — ยกเครือข่าย UI ใหม่ทั้งหมด เพิ่มประสิทธิภาพความเร็ว แท็บ Navigation, Transcript Timeline และ Settings
  • NEW Custom Template Manager — สามารถสร้าง สรุป และแก้ไขรูปแบบ AI Meeting Notes ตามความต้องการขององค์กรได้เอง
  • NEW Multiple Audio Input Handling — รองรับการดึงเสียงทั้งจาก Mic และ System Loopback Audio ช่วยให้ถอดเสียงได้คมชัดทุกฝั่งผู้พูด
