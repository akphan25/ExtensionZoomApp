ประวัติการอัปเดต

ทุกฟีเจอร์ใหม่ การแก้ไขบั๊ก และการปรับปรุงของ Zoom Meeting Transcriber By OATz เรียงตามเวอร์ชันล่าสุด

← กลับหน้าแรก
ฟีเจอร์ใหม่ แก้ไข / ปรับปรุง Breaking change
v1.1.0 Latest 20 กรกฎาคม 2026
About Tab & Onboarding
  • เพิ่มแท็บ About — แสดงเวอร์ชัน, ข้อมูลความเป็นส่วนตัว และทางเข้าคู่มือการใช้งานในที่เดียว
  • ย้ายปุ่มช่วยเหลือ จากแถบบนไปไว้ในแท็บ About ให้ UI สะอาดขึ้น
  • First-run onboarding — หน้าแนะนำการใช้งานสำหรับผู้ใช้ใหม่ครั้งแรกที่เปิด extension
v1.0.1 19 กรกฎาคม 2026
Review Fixes
  • ความแม่นยำของ prompt — แก้ transcription prompt ภาษาไทย/อังกฤษ, เก็บคำตอบรับสั้นที่เป็นคำพูดจริง และเน้น highlights ใน AI notes
  • แก้ปัญหา race condition — serialize การเขียนข้อมูลประชุมและ pending audio ป้องกันข้อมูลชนกัน
  • Export และ Copy ที่ตรงไปตรงมา — ตัดตัวเลือกที่ไม่ได้แปลงข้อมูลจริงออก เหลือ Plain text / Markdown / Slack
  • Chrome-only capture path — ลบโค้ด fallback เก่าที่ใช้งานไม่ได้จริงออก
v1.0.0 30 พฤษภาคม 2026
Stable 1.0 Release
  • Preact/Tailwind popup ใหม่ทั้งหมด — compact header, tab nav, transcript timeline, notes/settings/history และ export modal
  • Template Manager — สร้าง/แก้/ลบ custom template, ตั้ง default และ fallback อัตโนมัติ
  • Appearance themes — เลือก System / Light / Dark ได้ใน Settings
  • แก้ highlight anchoring — bookmark จุดสำคัญเกาะกับ transcript turn ที่ถูกต้องเสมอ
  • เพิ่ม automated test coverage ด้วย Vitest ครอบคลุมทุกโมดูลหลัก
v0.9.9 – v0.9.12 28–29 พฤษภาคม 2026
Rate-Limit Guardrails & Multi-mode Capture
  • เลือก Recording source ได้ 3 แบบ — Zoom audio + Mic, Zoom audio only, Mic only พร้อมจำค่าไว้
  • Gemini rate-limit gate — จำกัดคำขอไม่เกิน 10 ครั้ง/60 วินาที รอดจาก service worker restart
  • Retryable audio buffering — เสียงที่ถอดไม่สำเร็จเพราะเน็ตหลุดถูกเก็บไว้ replay อัตโนมัติ ไม่ทิ้งเสียง
  • Bundled Gemini SDK เข้ากับ service worker โดยตรง
v0.9.5 – v0.9.8 21 พฤษภาคม – 26 พฤษภาคม 2026
Transcript Hardening & Speaker Fingerprinting
  • Experimental MFCC speaker fingerprinting — ช่วยให้ label ผู้พูดเสถียรขึ้นภายในการประชุมเดียวกัน แบบ local-only ไม่ส่งข้อมูลออกนอกเครื่อง
  • Prompt injection guard — ครอบข้อมูลจาก transcript/notes ด้วย XML data block ทั้งหมด
  • Repetition loop guard — ตัดข้อความที่โมเดลวนซ้ำผิดปกติออกอัตโนมัติ
  • Corporate Thai-English Glossary preset — คำทับศัพท์ที่ใช้บ่อยในออฟฟิศไทย 16 คำ พร้อมใช้ทันที
  • เปลี่ยนชื่อ template variable — {notes}, {highlights}, {transcript} เปลี่ยนเป็น {notes_block}, {highlights_block}, {transcript_block}
v0.9.0 – v0.9.4 19–21 พฤษภาคม 2026
AI Enhancement
  • AI Notes Templates — 6 template สำเร็จรูป (General, 1:1, Sprint, Client, Brainstorm, Retro) พร้อมสร้าง custom เอง
  • Highlight / Mark moment — bookmark จุดสำคัญระหว่างประชุมด้วยคีย์ลัด Cmd/Ctrl+Shift+M
  • Multi-language Notes — Auto / Thai / English / Both
  • Follow-up Q&A — ถามคำถามต่อเนื่องจาก transcript พร้อม timestamp citation
  • เลือกไมค์และ echo cancellation — ลดเสียงลำโพงวนกลับเข้าไมค์เมื่ออัดพร้อมกัน
v0.6.0 – v0.8.0 19 พฤษภาคม 2026
Meeting History & Export Expansion
  • History tab — ค้นหา, เรียงลำดับ, filter และจัดการประชุมเก่าได้ทั้งหมด
  • Multi-format export — Markdown, Plain text, JSON, SRT subtitle พร้อมรูปแบบพิเศษสำหรับ Google NotebookLM
  • Multi-meeting storage — รองรับหลายการประชุมพร้อม custom vocabulary และ speaker alias
  • Bulk operations — เลือกหลายประชุมเพื่อลบ/export พร้อมดู storage usage
v0.1.0 – v0.5.0 18–19 พฤษภาคม 2026
Foundation & Thai-optimized Transcription
  • เปิดตัวครั้งแรก — MV3 extension จับเสียงไมโครโฟนและส่งไป Gemini API
  • Chrome tab capture — เพิ่ม offscreen document จับเสียงจากแท็บ Zoom โดยตรงผ่าน Web Audio API
  • Thai-first prompt — ถอดเสียงไทยตามหลักการสะกดมาตรฐาน ไม่ใช่ phonetic karaoke
  • Tab-based UI — แยก Transcript / Notes / Settings พร้อม status pill แสดงสถานะอัดเสียง
  • Retry + backoff สำหรับ Gemini API และ chunk queue แบบ serialize
